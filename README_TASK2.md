## Задание 2

Реализовано:

* `ru.csc.bdse.app.Application` - HTTP-сервер для телефонной книги версии 1.0.
  Требует HTTP-сервера ноды по адресу в переменной среды `KVNODE_URL` и строчки `1.0` в `PHONEBOOK_VERSION`.
* Формат сообщений хранится в `bdse-app/src/main/proto/pb10/phone_book.proto`.
* Для версии 1.1 предполагается скопировать папку `pb10` в `pb11` и подредактировать.
  Например, поля можно делать `repeated` с двухсторонней совместимостью в разумных пределах.
